## CI/CD and Deployment

This project deploys through GitHub Actions + Vercel.

- Pull requests to `master`: Preview deployment
- Pushes to `master`: Production deployment

Required GitHub repository secrets:

- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`
